# GYM - Gym Management System

GYM is a comprehensive, full-stack gym management system designed to streamline operations for fitness centers and sports academies. Built with modern web technologies, it offers an intuitive interface for managing students, tracking payments, sending notifications, and generating insightful reports.

![GYM Dashboard](https://placeholder.com/path-to-your-dashboard-screenshot.png)

## 🚀 Features

- **Student Management**: 
  - Add, edit, and delete student profiles
  - Track student attendance and progress
  - Manage student memberships and subscriptions
- **Payment Tracking**: 
  - Monitor monthly fees and payment status
  - Generate invoices automatically
  - Send payment reminders
- **WhatsApp Integration**: 
  - Send automated payment reminders via WhatsApp
  - Broadcast announcements to all or selected students
  - Schedule messages for future delivery
- **Analytics Dashboard**: 
  - Visualize key metrics such as total students, active memberships, and revenue
  - View yearly revenue chart with monthly breakdown
  - Track student retention and churn rates
- **Report Generation**: 
  - Create customizable PDF reports for financial summaries, student lists, and more
  - Export data in various formats (CSV, Excel)
- **Dark Mode**: 
  - Sleek, modern UI with a dark theme for reduced eye strain
  - Automatically switches based on system preferences or manual toggle

## 🛠️ Technologies Used

- **Frontend**: Next.js 13, React 18, Tailwind CSS
- **Backend**: Node.js, SQLite
- **State Management**: React Hooks
- **Charts**: Chart.js
- **PDF Generation**: jsPDF
- **WhatsApp Integration**: Twilio API
- **Styling**: Custom CSS with glassmorphism effects
- **Icons**: Lucide React
